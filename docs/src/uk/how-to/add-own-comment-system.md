# Add own comment system

Suppose you're not happy with the default commenting system and you decide to integrate an outside service.

Виберіть і увімкніть один з віджетів коментарів:

- Disqus
- FacebookComments
- Giscus
- VkComments

Change plugin settings to suit your needs, then go to _Portal => Settings => Pages and blocks_ area, look for the option labelled **Show page comments** and select your desired variant. Нарешті, збережіть налаштування.
