# Миграция на Light Portal

Переход на новый портал — важный шаг. Это руководство поможет вам перенести контент с других порталов для SMF на Light Portal.

## Подготовка

### Резервное копирование

Перед началом миграции **обязательно** сделайте полный бэкап:

- База данных форума
- Файлы форума (`Themes`, `Sources`)

:::warning Внимание

Никогда не начинайте миграцию на рабочем форуме без предварительного тестирования на локальном сервере или тестовом сайте.

:::

### Анализ текущего контента

Составьте список того, что нужно перенести:

- Блоки
- Страницы
- Категории

:::info Примечание

Поддерживается импорт только блоков и страниц со следующими типами контента: PHP/HTML/BBCode. Остальные типы блоков вам придется создать вручную.

:::

### Удаление предыдущего портала

Оставьте таблицы, созданные предыдущим порталом, в базе данных — они нужны для импорта.

## Миграция с TinyPortal

1. Установите и активируйте плагин TinyPortalMigration
2. Перейдите в нужный раздел — «Блоки», «Страницы» или «Категории», затем выберите пункт «Импорт из TinyPortal»

## Миграция с EhPortal (SimplePortal)

1. Установите и активируйте плагин EhPortalMigration
2. Перейдите в нужный раздел — «Блоки», «Страницы» или «Категории», затем выберите пункт «Импорт из EhPortal»

## Миграция с EzPortal

1. Установите и активируйте плагин EzPortalMigration
2. Перейдите в нужный раздел — «Блоки» или «Страницы», затем выберите пункт «Импорт из EzPortal»
