# Ενημέρωση...

Εάν το αρχείο καταγραφής αλλαγών δεν περιέχει συγκεκριμένες οδηγίες ενημέρωσης για την πιο πρόσφατη έκδοση, μπορείτε να κάνετε ενημέρωση εξαγάγοντας τους καταλόγους `Themes` και `Sources` στη ρίζα του φόρουμ σας, αντικαθιστώντας τα υπάρχοντα αρχεία.

Ωστόσο, για μια καθαρή εγκατάσταση, συνιστούμε να απεγκαταστήσετε πρώτα την τρέχουσα έκδοση.

:::info Σημειώσεις

Εάν η νέα έκδοση μπορεί να εγκατασταθεί χωρίς να αφαιρεθεί η προηγούμενη, θα εμφανιστεί το κουμπί "Αναβάθμιση" αντί για το κουμπί "Εγκατάσταση":

![Updating](upgrade.png)

:::
