# Dodaj lasten sistem komentarjev

Recimo, da nisi zadovoljen s privzetim sistemom komentarjev in se odločiš integrirati zunanjo storitev.

Izberi in omogoči enega od vtičnikov za komentarje:

- Disqus
- FacebookComments
- Giscus
- VkComments

Spremeni nastavitve vtičnika glede na svoje potrebe, nato pojdi v območje _Portal => Nastavitve => Strani in bloki_, poišči možnost z oznako **Prikaži komentarje** na strani in izberi želeno različico. Na koncu shrani svoje nastavitve.
